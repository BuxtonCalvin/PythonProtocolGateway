## Hardware
1. USB to RS485 Adapter & RJ45 Ethernet cable ( or 3 wires ) 
2. Connect RJ45 ethernet cable to an available RS485/Modbus port:
3. Connect appropriate wires to USB RS485 Adapter. see sol ark documentation
